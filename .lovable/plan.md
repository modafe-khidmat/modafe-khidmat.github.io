
# Mode Stack — Company Website

## Overview
A sleek, modern landing page for **Mode Stack** showcasing digital solutions and automation products. Static site with external Stripe Checkout links — no backend needed.

---

## Design Direction
- **Style**: Dark hero section transitioning to a clean white body — inspired by Stripe/Linear aesthetics
- **Typography**: Clean, modern sans-serif with strong hierarchy
- **Color palette**: Dark navy/black hero, white content area, Stripe blurple (#6772e5) for CTAs
- **Animations**: Subtle fade-in effects on scroll for polish

---

## Page Structure

### 1. Sticky Navigation
- Logo/brand name "Mode Stack" on the left
- Navigation links: **Home** and **Products**
- Compact, transparent header that becomes solid on scroll
- Fully responsive with mobile hamburger menu

### 2. Hero Section (Dark)
- Bold headline: **"Building the Future of Automation"**
- Subtext emphasizing digital solutions and automation
- Primary CTA button ("Explore Products") that smooth-scrolls to the product showcase
- Subtle background gradient or geometric pattern for visual interest

### 3. Product Showcase (White/Light)
- Section heading introducing the two core products
- **2-column grid** (stacks on mobile) with product cards:

  **Card A — Automated License System**
  - Icon/visual element
  - Title and description: Secure Node.js based licensing for offline/online activation
  - "Buy Now" button (Stripe blurple, links to `https://buy.stripe.com/placeholder`)

  **Card B — E-Commerce AI Agent**
  - Icon/visual element
  - Title and description: Automated price tracking and platform matching
  - "Buy Now" button (Stripe blurple, links to `https://buy.stripe.com/placeholder`)

### 4. Footer
- "Mode Stack" branding with copyright
- Minimal and clean

---

## Technical Approach
- Fully static — no backend, no database
- External Stripe Checkout links for payments (placeholder URLs)
- Fully responsive for mobile and desktop
- Smooth scroll behavior for in-page navigation
